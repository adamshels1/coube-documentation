# Demand Forecasting — Прогноз спроса на перевозки

**Приоритет:** P1
**Компоненты:** Backend (ML-модель), Frontend (дашборд), Mobile (уведомления)

## Описание

ML-система прогнозирования спроса на перевозки по направлениям, типам грузов и временным периодам. Помогает перевозчикам планировать загрузку, а заказчикам — выбирать оптимальное время.

## Модули

### 1. Прогноз объёма заявок

**Гранулярность:**
- По направлениям (пары регионов)
- По типам грузов
- По неделям (горизонт 4 недели)

**Алгоритм:**
- **Baseline:** Prophet (Facebook) — хорошо работает с сезонностью и праздниками
- **Advanced:** LightGBM с lag-features + календарные фичи

**Features:**
- Исторический объём заявок (lag 1-52 недели)
- День недели, неделя года
- Праздники Казахстана (Наурыз, Новый Год, День Конституции и др.)
- Сезон (посевная, уборочная, строительный сезон)
- Тип груза
- Регион отправления/назначения
- Средняя цена за период (прокси спроса)

**Данные:**
- `applications.transportation` — дата создания, тип груза
- `applications.cargo_loading` — регион загрузки/разгрузки
- `applications.transportation_cost` — стоимость

### 2. Прогноз цен

**Что показывает:** Ожидаемый диапазон цен на перевозки по направлениям на ближайшие недели.

**Алгоритм:** Quantile Regression (предсказание 10-го, 50-го, 90-го перцентилей цены)

**Features:**
- Прогнозируемый объём заявок (из модуля 1)
- Исторические цены по направлению
- Сезонность
- Доступность транспорта в регионе

### 3. Рекомендации для перевозчиков

На основе прогнозов формируются персонализированные рекомендации:

```
"На следующей неделе ожидается рост заявок Алматы→Астана на 30%.
Рекомендуем перенаправить 2 машины в Алматинскую область.
Ожидаемая цена: 420-480 тыс. тг (рост 12% к текущей неделе)"
```

### 4. Рекомендации для заказчиков

```
"Сейчас высокий сезон на направлении Шымкент→Алматы.
Если можете подождать 2 недели, цена снизится на ~15%.
Рекомендуемая цена сейчас: 380 тыс. тг"
```

Персонализированные рекомендации для перевозчика/заказчика на основе его истории.

## UI-компоненты

### Frontend — Дашборд прогнозов
- График прогноза спроса по направлениям (линейный + confidence interval)
- Карта с прогнозом спроса по регионам
- Таблица ТОП-направлений с трендами (стрелки вверх/вниз)
- Раздел персональных рекомендаций

### Mobile — Push-рекомендации
- Еженедельные дайджесты для перевозчиков: "Топ направления на этой неделе"
- Уведомление при резком росте спроса на направлении перевозчика

### Frontend — Виджет при создании заявки
- "Спрос сейчас: высокий. Рекомендуем цену: 430 тыс. тг"
- Мини-график тренда спроса на выбранном направлении

## Требования к данным

- **Минимум:** 6 месяцев истории перевозок для базового прогноза
- **Оптимум:** 12+ месяцев для учёта годовой сезонности
- **Обновление модели:** еженедельно
- **Обновление прогноза:** ежедневно

## Метрики успеха

- MAPE прогноза объёма: <25% на горизонте 1 неделя, <35% на 4 недели
- MAPE прогноза цены: <20%
- Engagement: >40% перевозчиков просматривают прогнозы
- Улучшение загрузки транспорта: +10% (снижение простоев)
