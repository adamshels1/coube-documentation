## 8) Исходный код: вес, языки, версия

### Backend (coube-backend)
- **Язык**: Java 23

- **Файлов**: 1,152 .java
- **Размер**: ~15-20 МБ
- **Технологии**: PostgreSQL+PostGIS, Keycloak OAuth2, MinIO, Redis, Flyway, Kalkan ЭЦП

### Frontend Web (coube-frontend)
- **Язык**: TypeScript
- **Фреймворк**: Vue.js 3.5.13
- **Размер**: ~5-10 МБ
- **Технологии**: Pinia, Vue Router, Axios, Vue i18n (4 языка: ru, en, kk, zh)

- **Фреймворк**: React Native 0.81.5

- **Технологии**: Zustand, React Navigation, Yandex Maps, Firebase Push, Background Geolocation

- **Схем**: 7
