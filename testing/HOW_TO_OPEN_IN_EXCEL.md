# Как открыть чек-лист в Google Sheets

## Импорт файла в Google Sheets

1. Откройте [Google Sheets](https://sheets.google.com)
2. Нажмите **Файл** → **Импортировать**
3. Перейдите на вкладку **Загрузка**
4. Перетащите файл `manual-testing-checklist.csv` или выберите через кнопку
5. В настройках импорта:
   - **Тип импорта:** `Заменить текущий лист` или `Создать новую таблицу`
   - **Разделитель:** `Определить автоматически` или `Запятая`
6. Нажмите **Импортировать данные**

**Примечание:** Google Sheets автоматически определяет кодировку UTF-8, поэтому русский текст будет отображаться корректно.

---

## Рекомендации после открытия в Google Sheets

1. **Закрепите первую строку (заголовки):**
   - Выделите строку 1 → **Вид** → **Закрепить** → **1 строку**
   - Это позволит видеть заголовки при прокрутке

2. **Включите фильтры:**
   - Выделите строку заголовков
   - **Данные** → **Создать фильтр**
   - Появятся кнопки фильтрации в каждом столбце

3. **Настройте ширину столбцов:**
   - Выделите все столбцы → правой кнопкой → **Изменить размер столбцов A-H** → выберите **Автоподбор**
   - Или дважды кликните на границу между заголовками столбцов

4. **Предоставьте доступ команде:**
   - Нажмите кнопку **Настроить доступ** (правый верхний угол)
   - Добавьте email участников команды тестирования
   - Выберите уровень доступа: "Редактор" (для заполнения результатов)

## Структура чек-листа (столбцы)

| Столбец | Описание |
|---------|----------|
| № | Порядковый номер теста |
| Модуль | Название модуля системы |
| Функция | Конкретная функция для проверки |
| Роль пользователя | Кто выполняет действие |
| Описание проверки | Что нужно проверить |
| Критичность | Уровень важности теста |
| Результат | Заполняется при тестировании (Pass/Fail/Blocked/Skip) |
| Комментарий | Дополнительные заметки, номера багов |

---

## Готовые формулы для подсчета результатов

После заполнения столбца "Результат" (столбец G) можно добавить формулы в конце таблицы (например, строка 105):

| Метрика | Формула |
|---------|---------|
| Всего тестов | `=COUNTA(A2:A102)` |
| Пройдено | `=COUNTIF(G2:G102,"Pass")` |
| Провалено | `=COUNTIF(G2:G102,"Fail")` |
| Заблокировано | `=COUNTIF(G2:G102,"Blocked")` |
| Пропущено | `=COUNTIF(G2:G102,"Skip")` |
| Процент успеха | `=COUNTIF(G2:G102,"Pass")/COUNTA(G2:G102)*100` |

**Пример использования:**
1. В ячейку A105 введите: `Всего тестов:`
2. В ячейку B105 введите формулу: `=COUNTA(A2:A102)`
3. Повторите для остальных метрик

## Преимущества использования Google Sheets

✅ **Совместная работа в реальном времени** - несколько тестировщиков могут работать одновременно

✅ **Автосохранение** - все изменения сохраняются автоматически

✅ **История изменений** - можно посмотреть, кто и когда внес изменения

✅ **Не требует установки** - работает в любом браузере

✅ **Комментарии** - можно оставлять комментарии к конкретным ячейкам
