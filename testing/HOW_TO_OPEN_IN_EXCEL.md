# Как открыть чек-лист в Excel

## Способ 1: Microsoft Excel (Windows/Mac)

### Для корректного отображения русского текста:

1. Откройте Microsoft Excel
2. Перейдите в меню: **Файл** → **Импорт** → **CSV файл**
3. Выберите файл `manual-testing-checklist.csv`
4. В диалоге импорта:
   - **Кодировка файла:** выберите `UTF-8`
   - **Разделитель:** выберите `Запятая (,)`
   - **Текстовый разделитель:** `"` (кавычки)
5. Нажмите **Готово**

### Альтернативный способ (быстрый):
- Просто перетащите файл `manual-testing-checklist.csv` в окно Excel
- Excel автоматически определит кодировку

---

## Способ 2: Google Sheets (онлайн)

1. Откройте [Google Sheets](https://sheets.google.com)
2. Нажмите **Файл** → **Импортировать**
3. Перейдите на вкладку **Загрузка**
4. Перетащите файл `manual-testing-checklist.csv` или выберите через кнопку
5. В настройках импорта:
   - **Тип импорта:** `Заменить текущий лист` или `Создать новую таблицу`
   - **Разделитель:** `Определить автоматически` или `Запятая`
6. Нажмите **Импортировать данные**

---

## Способ 3: LibreOffice Calc (бесплатно)

1. Откройте LibreOffice Calc
2. Перейдите в меню: **Файл** → **Открыть**
3. Выберите файл `manual-testing-checklist.csv`
4. В диалоге импорта:
   - **Кодировка:** `Unicode (UTF-8)`
   - **Разделитель полей:** отметьте `Запятая`
   - **Разделитель строк:** `"`
5. Нажмите **ОК**

---

## Способ 4: Открытие прямо из файлового менеджера

### На macOS:
- Двойной клик на файл → откроется в Numbers (встроенное приложение Apple)
- Или правой кнопкой → **Открыть с помощью** → **Microsoft Excel**

### На Windows:
- Двойной клик на файл → откроется в Excel (если установлен)
- Или правой кнопкой → **Открыть с помощью** → выберите приложение

## Возможные проблемы и решения

### Проблема 1: Русский текст отображается как "крякозябры"
**Решение:** 
- Откройте файл через импорт (см. Способ 1) с явным указанием кодировки UTF-8
- Или используйте Google Sheets - он автоматически определяет кодировку

### Проблема 2: Все данные в одном столбце
**Решение:**
- При импорте укажите разделитель: **Запятая (,)**
- В Excel: **Данные** → **Текст по столбцам** → выберите разделитель "Запятая"

### Проблема 3: Нет Excel/Office
**Решение:**
- Используйте бесплатные альтернативы:
  - Google Sheets (онлайн, не требует установки)
  - LibreOffice Calc (бесплатный, https://www.libreoffice.org)
  - Apple Numbers (встроено в macOS)

## Рекомендации после открытия

1. **Закрепите первую строку (заголовки):**
   - Excel/Sheets: выделите строку 1 → **Вид** → **Закрепить строки**
   - Это позволит видеть заголовки при прокрутке

2. **Установите фильтры:**
   - Выделите строку заголовков
   - **Данные** → **Фильтр**
   - Появятся кнопки фильтрации в каждом столбце

3. **Настройте ширину столбцов:**
   - Дважды кликните на разделитель между столбцами
   - Или выделите все столбцы → правой кнопкой → **Подогнать по содержимому**

4. **Сохраните как Excel файл (опционально):**
   - **Файл** → **Сохранить как** → выберите формат `.xlsx`
   - Это позволит сохранить форматирование, фильтры и другие настройки

## Структура чек-листа (столбцы)

| Столбец | Описание |
|---------|----------|
| № | Порядковый номер теста |
| Модуль | Название модуля системы |
| Функция | Конкретная функция для проверки |
| Роль пользователя | Кто выполняет действие |
| Описание проверки | Что нужно проверить |
| Критичность | Уровень важности теста |
| Результат | Заполняется при тестировании (Pass/Fail/Blocked/Skip) |
| Комментарий | Дополнительные заметки, номера багов |

---

## Готовые формулы для подсчета результатов

После заполнения столбца "Результат" можно добавить формулы в конце таблицы:

```excel
Всего тестов: =COUNTA(A2:A102)
Пройдено: =COUNTIF(G2:G102,"Pass")
Провалено: =COUNTIF(G2:G102,"Fail")
Заблокировано: =COUNTIF(G2:G102,"Blocked")
Пропущено: =COUNTIF(G2:G102,"Skip")
Процент успеха: =COUNTIF(G2:G102,"Pass")/COUNTA(G2:G102)*100
```
