**Отчеты для Перевозчика**

**1. Реестр завершенных заявок и перевозок**

**Excel / Таблица**  
\| № заявки \| № рейса \| Заказчик \| Груз \| Вес (т) \| Маршрут \|
Статус \| Дата создания \| Дата исполнения \| Стоимость (Т) \| Пробег
(км) \|

| **Поле**        | **Тип**  | **Описание**                  | **Единицы** |
|-----------------|----------|-------------------------------|-------------|
| № заявки        | string   | Уникальный ID заявки          | --          |
| № рейса         | string   | Уникальный ID рейса           | --          |
| Заказчик        | string   | Название компании-заказчика   | --          |
| Груз            | string   | Наименование груза            | --          |
| Вес             | float    | Вес груза                     | тонны       |
| Маршрут         | string   | Откуда → Куда                 | --          |
| Статус          | enum     | Новый/В пути/Завершен/Отменен | --          |
| Дата создания   | datetime | Дата создания заявки          | --          |
| Дата исполнения | datetime | Фактическая дата завершения   | --          |
| Стоимость       | float    | Стоимость перевозки           | Т           |
| Пробег          | float    | Длина маршрута                | км          |

**UI-дизайн**

- Таблица с цветовой маркировкой статусов (новая/в
  пути/завершена/отмена).

- Фильтры сверху (дата, заказчик, статус).

**Бэк**

- Источники: orders, trips.

- Онлайн обновление при смене статуса.

- Сводка (carrier_shipments_registry) обновляется раз в сутки.

**8. Гео-аналитика маршрутов**

**Excel / Таблица**  
\| № рейса \| Маршрут \| Груз (т) \| Пробег (км) \| Стоимость (Т) \|
Цена/т \| Цена/т/км \|

| **Поле**  | **Тип** | **Описание**                 | **Единицы** |
|-----------|---------|------------------------------|-------------|
| № рейса   | string  | ID рейса                     | --          |
| Маршрут   | string  | Откуда → Куда                | --          |
| Груз      | float   | Вес груза                    | т           |
| Пробег    | float   | Расстояние маршрута          | км          |
| Стоимость | float   | Общая стоимость рейса        | Т           |
| Цена/т    | float   | Стоимость на 1 тонну         | Т           |
| Цена/т/км | float   | Стоимость на 1 тонну за 1 км | ₽/т/км      |

- Карта с маршрутами.

**Бэк**

- Источники: GPS, trips.

- Формулы:

  - Цена/т = Сумма / тоннаж

  - Цена/т/км = Сумма / тоннаж / пробег

- Обновление раз в сутки.
