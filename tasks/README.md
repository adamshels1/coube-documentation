---

## НДС — краткое описание

Добавить в форму заявки блок расчёта НДС с двумя режимами:
- **Добавить НДС к сумме** — введённая сумма = база без НДС
- **Выделить НДС из суммы** — введённая сумма уже включает НДС (16%)

Изменения затрагивают: **Backend + Frontend + Mobile**

## ИС ЭСФ — краткое описание

Интеграция с государственной системой электронных счетов-фактур КГД РК.
Пользователи смогут выставлять ЭСФ прямо из раздела финансовых документов COUBE.

Изменения затрагивают: **Backend + Frontend** (Mobile — только отображение)

---

## Зависимости между фичами

```
НДС в заявке (Transportation)
    ↓ данные НДС попадают в
Счёт на оплату (Invoice)
    ↓ основание для
ЭСФ в ИС ЭСФ
```

**НДС нужно реализовать раньше ИС ЭСФ**, потому что корректные суммы с НДС в счёте — это входные данные для ЭСФ.
