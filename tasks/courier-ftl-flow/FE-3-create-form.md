# FE-3 — Форма создания: ввод точек доставки для COURIER_DELIVERY

**Статус:** 🔴 Критично  
**Оценка:** 1 день  
**Зависит от:** FE-1, FE-2  
**Файл:** `coube-frontend/src/views/Transportations/TransportationCreate/TransportationsCreatePage.vue`

---

## Описание

При выборе типа `COURIER_DELIVERY` в форме создания заявки показать блок для ручного ввода точек маршрута. По умолчанию в FTL 1-2 точки, для курьерки может быть 2-50+.

---

## Чеклист

### Условный блок при выборе COURIER_DELIVERY

- [ ] Показывать блок ввода точек только при `transportationType === COURIER_DELIVERY`
- [ ] Скрывать поля специфичные для FTL (тип кузова, вес груза и т.п.) при COURIER_DELIVERY

### Компонент точки маршрута (можно переиспользовать из CargoLoading)

Каждая точка содержит:
- [ ] Тип точки: `LOADING` (погрузка) / `UNLOADING` (разгрузка)
- [ ] Адрес (текстовое поле + интеграция с картой Яндекс)
- [ ] Координаты (lat/lon) — заполняются автоматически при выборе адреса на карте
- [ ] Дата и время (опционально)
- [ ] Контактное лицо (имя)
- [ ] Контактный телефон
- [ ] Комментарий
- [ ] Требуется SMS подтверждение (чекбокс `isSmsRequired`)
- [ ] Требуется фото (чекбокс `isPhotoRequired`)

### Управление списком точек

- [ ] Кнопка "Добавить точку" — добавляет новую точку в список
- [ ] Возможность удалить точку (кроме первой и последней)
- [ ] Drag-and-drop для изменения порядка (опционально, можно добавить позже)
- [ ] Нумерация точек (1, 2, 3...)

### Валидация

- [ ] Минимум 1 точка типа `LOADING`
- [ ] Минимум 1 точка типа `UNLOADING`
- [ ] Адрес обязателен для каждой точки
- [ ] Контактный телефон обязателен для точек типа `UNLOADING`

### Стоимость маршрута

- [ ] Поле для ввода стоимости маршрута целиком (как в FTL)
- [ ] Валюта, отсрочка платежа, авансовый платёж
- [ ] Чекбокс "Использовать факторинг"

### API запрос

- [ ] `POST /api/v1/customer/transportation/complete`
  ```json
  {
    "transportationType": { "code": "COURIER_DELIVERY" },
    "cargoLoadings": [
      {
        "loadingType": { "code": "LOADING" },
        "address": "...",
        "point": { "lon": 76.85, "lat": 43.22 },
        "contactName": "...",
        "contactNumber": "...",
        "isSmsRequired": false,
        "isPhotoRequired": false
      },
      {
        "loadingType": { "code": "UNLOADING" },
        ...
      }
    ],
    "costInfo": { "cost": 5000, ... }
  }
  ```
- [ ] Проверить что запрос успешно создаёт Transportation с нужными полями

### Проверка

- [ ] `npm run type-check` — нет TS ошибок
- [ ] `npm run lint` — нет ошибок линтера
