# FE-4 — Список заявок: фильтр и карточка COURIER_DELIVERY

**Статус:** 🟡 Важно  
**Оценка:** 3ч  
**Зависит от:** FE-1, FE-2  
**Файл:** `coube-frontend/src/views/Applications/ApplicationsPage.vue`

---

## Описание

Добавить поддержку фильтрации и отображения заявок типа `COURIER_DELIVERY` в списках заказчика и перевозчика.

---

## Чеклист

### Список заявок заказчика

- [ ] Добавить `COURIER_DELIVERY` в фильтр по типу перевозки
- [ ] Карточка заявки показывает тип (бейдж/тег "Курьерская доставка")
- [ ] Карточка курьерской заявки показывает количество точек маршрута
- [ ] Статусы курьерской заявки отображаются корректно (те же что FTL)

### Список заявок перевозчика

- [ ] `COURIER_DELIVERY` попадает в список при фильтре `transportationType=COURIER_DELIVERY`
- [ ] Перевозчик видит курьерские заявки в статусе `CREATED` (новые, доступные для отклика)
- [ ] Фильтр по типу работает через `POST /api/v1/executor/transportations` с `transportationType: 'COURIER_DELIVERY'`

### Визуальное отличие карточки

- [ ] Иконка или цветовой маркер для `COURIER_DELIVERY`
- [ ] Показывать количество точек доставки (а не вес/объём как в FTL)
- [ ] i18n ключ для подписи типа берётся из `transportationTypesDictionary`

### Проверка

- [ ] Фильтр работает: заявки `COURIER_DELIVERY` видны при выборе этого типа
- [ ] Фильтр не ломает отображение FTL заявок
